# 📑 Project: Dopamine Explorer v2 (Vintage Dashboard & Serverless Cloud Builder)

이 프로젝트는 **전 세계 검증된 예능 소재를 발굴**하고, 파이썬 기반의 **영상 자동 전처리**를 거쳐, **최종 편집은 캡컷(CapCut)에서 5분 만에 끝낼 수 있도록** 돕는 100% 무설치 서버리스 올인원 쇼츠 제작 시스템입니다.

개인 PC에 어떠한 파이썬도, 요리 도구(MoviePy, edge-tts)도 설치할 필요가 전혀 없습니다! 
사용자는 오직 **GitHub Pages 전용 웹 대시보드 주소**로 24시간 언제 어디서나 접속하여 마우스 클릭 한 번으로 모든 영상을 스캔하고, 컷팅하고, FCP XML 및 가공 미디어가 담긴 1일 한정 만료 ZIP 압축 파일을 간편하게 다운로드받아 편집을 끝낼 수 있습니다.

## 🚀 3초 만에 깃허브 페이지(GitHub Pages) 활성화 및 접속 가이드

깃허브 저장소에 코드를 올려두셨다면, 단 3번의 클릭만으로 24시간 어디서나 상시 접속 가능한 고유 대시보드 인터넷 사이트가 즉시 오픈됩니다.

### **[1단계] GitHub Pages 기능 활성화하기**
1. 나의 깃허브 저장소 페이지([https://github.com/GoldSH69/DE](https://github.com/GoldSH69/DE))에 로그인해 접속합니다.
2. 상단 메뉴 탭에서 톱니바퀴 모양의 🛠️ **`Settings`** (설정) 아이콘을 클릭합니다.
3. 좌측 사이드바 메뉴 중간의 **`Code and automation`** 섹션 아래에 있는 🌐 **`Pages`** 메뉴를 클릭합니다.

### **[2단계] 배포 브랜치 지정하기**
1. 화면 중앙의 **`Build and deployment`** 섹션에서 **`Source`** 항목이 `Deploy from a branch`로 설정되어 있는지 확인합니다.
2. 바로 아래 **`Branch`** 드롭다운 메뉴를 클릭하여 `None` 대신 **`main`** 브랜치로 변경합니다.
3. 그 옆의 폴더 드롭다운은 기본값인 **`/(root)`** 그대로 두고, 우측의 💾 **`Save`** (저장) 버튼을 꾹 누릅니다.

### **[3단계] 전용 대시보드 사이트 접속하기**
1. 약 30초~1분 정도 기다리면, Settings -> Pages 페이지의 가장 상단에 초록색 체크마크와 함께 **"Your site is live at..."** 이라는 안내 문구와 함께 나만의 전용 인터넷 주소가 배출됩니다.
2. 주소 형식: **`https://GoldSH69.github.io/DE/`**
3. 이제 스마트폰, 아이패드, 외부 노트북 등 기기에 구애받지 않고 언제 어디서나 웹 브라우저 주소창에 이 링크만 치고 들어가면 웜 샌드 대시보드가 화사하게 열립니다!

## 💡 대시보드 실전 사용 요약 (All-in-One Workflow)

1. **설정 등록 (최초 1회)**:
   * 대시보드 우측 상단의 **[연동 키 설정]** 기어 아이콘을 누릅니다.
   * 내 GitHub Personal Access Token (PAT - repo 권한 체크 필수) 및 내 Gemini API Key(선택)를 붙여넣고 저장합니다. (브라우저 로컬 스토리지에만 안전 암호화 저장되어 유출 리스크가 0%입니다.)
2. **핫 예능 기간 스캔 및 영상 새 창 검증**:
   * 검색창에 스캔 키워드(예: "일본 예능 레전드")를 입력합니다.
   * 기간 필터 드롭다운에서 **[오늘 (1일)] / [이번 주 (1주)] / [이번 달 (30일)]** 을 용도에 맞게 선택한 뒤 [소재 스캔하기]를 누릅니다.
   * 벼락같이 정렬되어 나오는 비디오 카드의 제목이나 이미지를 마우스 클릭하면, **새 창(New Tab)으로 유튜브 원본 영상이 켜져서** 어떤 내용이고 조회수가 실제로 얼마나 높은지 편하게 직접 감상 및 확인합니다.
3. **원격 기동 및 1일 만료 ZIP 다운로드**:
   * 감상이 끝난 비디오 카드를 클릭해 **[제작 타겟 지정] 하이라이트**를 활성화합니다.
   * 우측 제어반 하단의 **[쇼츠 자동화 제작 기동 (GitHub Actions)]** 버튼을 가볍게 클릭합니다.
   * 대시보드 터미널에 깃허브 클라우드가 자동으로 배정되어 일을 시작하고 끝마치는 진행 상황이 실시간 스트리밍 송출됩니다.
   * 완료 성공 시, 터미널 밑에 동적으로 솟아오르는 **[완성본 ZIP 압축 아카이브 다운로드]** 버튼을 눌러 압축 파일만 내 PC에 쏙 받아옵니다!
   * 이 압축 파일은 저장 용량 관리를 위해 단 **1일(24시간)의 수명**만 설정되어 있어, 하루 뒤 깃허브가 알아서 깨끗이 흔적도 없이 자동 영구 청소해 줍니다. 깃허브 서버 용량 부담 걱정이 전혀 없습니다.

## 🛠️ 기술적 뼈대 및 모듈 구조

* **`index.html` / `style.css` / `app.js`**: 100% 무설치 정적 웹서버리스 프론트엔드로, 깃허브 페이지에 안전 구동되어 GitHub Actions API를 직접 다이렉트로 제어합니다.
* **`src/main.py` & `src/processor.py` 등**: 깃허브 가상 러너 컴퓨터 내부에서 은밀하고 정확하게 작동하여 영상을 고속 다운로드 및 MoviePy 컷팅, edge-tts 합성, FCP XML 빌드를 완수하는 백엔드 엔진입니다.
